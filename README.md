# 🌍 Satellite Image Analysis for Urban Growth & Climate Monitoring

## 📌 Overview
Urbanization is rapidly transforming landscapes, impacting environmental sustainability and climate systems. This project leverages **satellite imagery, machine learning, and data visualization** to monitor urban growth and analyze its environmental effects.

The system integrates:
- Remote sensing data (Sentinel & Landsat)
- Google Earth Engine (GEE)
- Machine Learning (Linear Regression)
- Interactive dashboard using Streamlit

---

## 🚀 Features

- 📡 Detect urban expansion over time
- 🌱 Analyze vegetation using NDVI
- 🌡 Monitor land surface temperature (LST)
- 📊 Predict future urban growth trends
- 🖥 Interactive dashboard for visualization

---

## 🧠 Methodology

### 1. Data Collection
- Sentinel-2 (high-resolution imagery)
- Landsat (historical data)
- Data fetched using Google Earth Engine

### 2. Preprocessing
- Cloud masking
- Image filtering by date & region
- Normalization

### 3. Feature Extraction
- NDVI Calculation:

- Land Surface Temperature (LST)

### 4. Machine Learning
- Linear Regression for prediction
- Random Forest for classification

### 5. Urban Growth Analysis
- Multi-temporal change detection
- Urban expansion quantification

### 6. Visualization
- Streamlit Dashboard
- Graphs and geospatial maps

---

## 📊 Results

- 📈 Significant urban expansion detected
- 🌿 Decline in vegetation (NDVI drop)
- 🌡 Increase in land surface temperature
- 🔮 Future predictions for 2030 & 2035
